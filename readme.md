# WP Clean Invoice Generator

A lightweight WordPress plugin that generates clean and simple invoices.

## Features
- Clean and professional invoice layout
- Lightweight and fast
- Easy integration with WordPress
- Printable invoices
- Mobile friendly

## Installation
1. Download or clone this repository
2. Upload the plugin folder to `/wp-content/plugins/`
3. Go to WordPress Admin → Plugins
4. Activate **WP Clean Invoice Generator**
4. Copy [woo_invoice_table] short code to you thankyou page created by elemetor or  default woocommerce thankyou page.



## Usage
After activating the plugin, you can generate invoices from your WordPress dashboard.
