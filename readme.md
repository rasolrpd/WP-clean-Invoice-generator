# WP Clean Invoice Generator

A lightweight WordPress plugin that generates clean and simple invoices.

## Features
- Clean and professional invoice layout
- Lightweight and fast
- Easy integration with WordPress
- Printable invoices
- Mobile friendly

## Installation
1. Download or clone this repository
2. Upload the plugin folder to `/wp-content/plugins/`
3. Go to WordPress Admin → Plugins
4. Activate **WP Clean Invoice Generator**
5. create a page and name it as"order-summary" (slog is important not hte tittle of the page) and copy [woo_invoice_table] to your page.
 



## Usage
After activating the plugin, now yoy will have clean invoice instead of the normal thankyou page order summary. you can also download or print the invoice on the page.
